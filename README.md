# Omniauth::EoCustom

Entrepreneurs' Organization Custom strategy implementation

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'omniauth-eo-custom'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omniauth-eo-custom
